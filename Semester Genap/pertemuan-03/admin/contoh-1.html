Halaman ini demo 3 hal: select dropdown (komponen input), navbar dropdown CSS-only, dan multi-tier dropdown + versi mobile pakai <details>.

1) Dropdown Box: <select> + <option>

Ini contoh dropdown untuk memilih kategori. Tanpa JavaScript, dropdown ini fokusnya untuk latihan input dan tampilan UI (bukan pindah halaman otomatis).

Tips: gunakan label biar jelas, dan pakai styling wrapper supaya dropdown terlihat rapi.
